# Aion Language Server (MVP)

Run `tools/aion_lsp.py` as a stdio server. It supports:
- Initialize
- Hover: shows the `#intent` goal
- Diagnostics: flags missing `#intent`

This is a starting point for richer features (go-to, completions, checker/nullflow diagnostics).
